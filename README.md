# 🎬 Auto TikTok & Reels Clipper

سكربت بايثون خفيف وسريع لتحويل الفيديوهات الأفقية الطويلة (مثل فيديوهات يوتيوب) إلى مقاطع قصيرة وعشوائية (بين 30 و 40 ثانية) بأبعاد عمودية (9:16) جاهزة للرفع على TikTok، Instagram Reels، و YouTube Shorts.

## 🚀 المميزات
- تقطيع عشوائي للفيديو لتجنب التطابق وحقوق الملكية.
- تحويل الأبعاد تلقائياً إلى 9:16 (Center Crop).
- تسمية المقاطع وتنظيمها تلقائياً في مجلد `output`.
- لا يتطلب إمكانيات جهاز خرافية للعمل.

## ⚙️ المتطلبات الأساسية
يجب أن يكون لديك البرامج التالية مثبتة على جهازك:
1. **Python 3.x**
2. **FFmpeg** (محرك معالجة الفيديوهات الأساسي)

---

## 🛠️ طريقة التثبيت والتشغيل

### 1️⃣ التشغيل على نظام لينكس (Kali / Ubuntu / Debian)
افتح الـ Terminal ونفذ الأوامر التالية بالترتيب:

```bash
# تحديث الحزم وتثبيت FFmpeg
sudo apt update
sudo apt install ffmpeg -y

# إنشاء بيئة افتراضية وتفعيلها (مهم لتجنب تعارض الحزم)
python3 -m venv venv
source venv/bin/activate

# تثبيت مكتبة moviepy الإصدار المتوافق
pip install moviepy==1.0.3

# تأكد من وضع ملف الفيديو باسم video.mp4 في نفس المجلد ثم شغل السكريبت
python3 auto_tiktok.py

```

### 2️⃣ التشغيل على نظام ويندوز (Windows)

1. قم بتحميل وتثبيت **Python** من الموقع الرسمي (تأكد من تفعيل خيار `Add Python to PATH` أثناء التثبيت).
2. قم بتحميل وتثبيت **FFmpeg** وإضافته إلى مسار النظام (Environment Variables PATH).
3. افتح موجه الأوامر (CMD) داخل مجلد السكريبت واكتب:

```cmd
pip install moviepy==1.0.3

```

4. ضع الفيديو باسم `video.mp4` بجوار السكريبت، ثم شغل الأداة:

```cmd
python auto_tiktok.py

```

### 3️⃣ التشغيل على الموبايل (Android عبر تطبيق Termux)

يمكنك تشغيل السكريبت مباشرة من هاتفك الأندرويد باستخدام محاكي الطرفية Termux:

1. قم بتحميل تطبيق **Termux** من متجر F-Droid (تجنب نسخة جوجل بلاي لأنها غير محدثة).
2. افتح التطبيق ونفذ الأوامر التالية لإعداد البيئة:

```bash
pkg update && pkg upgrade -y
pkg install python ffmpeg -y

```

3. اسمح لـ Termux بالوصول إلى ملفات الهاتف:

```bash
termux-setup-storage

```

4. قم بتثبيت المكتبة المطلوبة:

```bash
pip install moviepy==1.0.3

```

5. انقل السكريبت والفيديو `video.mp4` إلى مجلد داخل مسار Termux، ثم قم بتشغيله:

```bash
python auto_tiktok.py

```

## 📁 هيكل المجلد قبل التشغيل

تأكد أن المجلد الخاص بك يبدو هكذا قبل تشغيل السكريبت:

```text
/auto_tiktok
 ├── auto_tiktok.py
 └── video.mp4

```

بعد التشغيل، سيقوم السكريبت بإنشاء مجلد `output` يحتوي على المقاطع الجاهزة.
